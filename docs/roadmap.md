- [Token Metadata API](https://github.com/orgs/hirosystems/projects/11/views/4?filterQuery=-label%3Aicebox+-status%3A%22%E2%9C%85+Done%22+repo%3A%22hirosystems%2Ftoken-metadata-api%22)
