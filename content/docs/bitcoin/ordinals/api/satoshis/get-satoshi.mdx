<API>

<APIInfo method={"GET"} route={"/ordinals/v1/sats/{ordinal}"}>

Retrieves ordinal information for a single satoshi.

### Path parameters

<Property required={true} deprecated={false} name={"ordinal"} type={"integer"}>

Ordinal number that uniquely identifies a satoshi

<span>Minimum: `0`</span>

</Property>

| Status code | Description      |
| :---------- | :--------------- |
| `200`       | Default response |
| `400`       | Default response |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X GET "https://api.hiro.so/ordinals/v1/sats/2099999997689999"
```
