<API>

<APIInfo method={"GET"} route={"/ordinals/v1/sats/{ordinal}/inscriptions"}>

Retrieves all inscriptions associated with a single satoshi.
### Query Parameters

<Property required={false} deprecated={false} name={"offset"} type={"integer"}>

Result offset

<span>Minimum: `0`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>
### Path Parameters

<Property required={true} deprecated={false} name={"ordinal"} type={"integer"}>

Ordinal number that uniquely identifies a satoshi

<span>Minimum: `0`</span>

</Property>

| Status code | Description      |
| :---------- | :--------------- |
| `200`       | Default response |
| `400`       | Default response |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X GET "https://api.hiro.so/ordinals/v1/sats/2099999997689999/inscriptions"
```
