Clone the [Bitcoin Indexer repository](https://github.com/hirosystems/bitcoin-indexer) to your local machine and install via `cargo`.

```terminal
$ git clone https://github.com/hirosystems/bitcoin-indexer.git
$ cd bitcoin-indexer
$ cargo bitcoin-indexer-install
```
