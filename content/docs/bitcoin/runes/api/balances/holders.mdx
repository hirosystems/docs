<API>

<APIInfo method={"GET"} route={"/runes/v1/etchings/{etching}/holders"}>

Retrieves a paginated list of holders for a Rune.

### Query Parameters

<Property required={false} deprecated={false} name={"offset"} type={"integer"}>

Result offset

<span>Minimum:  `0`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

Results per page

<span>Minimum:  `1`</span>

<span>Maximum:  `60`</span>

</Property>

### Path Parameters

<Property required={true} deprecated={false} name={"etching"} type={"string"}>

Value can represent a Rune ID, Rune number, Rune name, or Rune name with spacers.

</Property>

| Status code | Description |
| :----------- | :----------- |
| `200` | Default Response |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X GET "https://api.hiro.so/runes/v1/etchings/ZZZZZFEHUZZZZZ/holders"
```
