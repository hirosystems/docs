<API>

<APIInfo method={"GET"} route={"/ordinals/v1/brc-20/tokens/{ticker}/holders"}>

Retrieves a list of holders and their balances for a specified BRC-20 token.

### Query parameters

<Property required={false} deprecated={false} name={"offset"} type={"integer"}>

Result offset

<span>Minimum: `0`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>

### Path parameters

<Property required={true} deprecated={false} name={"ticker"} type={"string"}>

</Property>

| Status code | Description      |
| :---------- | :--------------- |
| `200`       | Default response |
| `404`       | Default response |

</APIInfo>

<APIExample>

```bash title="curl"
curl -X GET "https://api.hiro.so/ordinals/v1/brc-20/tokens/string/holders"
```
