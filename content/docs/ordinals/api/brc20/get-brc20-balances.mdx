<API>

<APIInfo method={"GET"} route={"/ordinals/v1/brc-20/balances/{address}"}>

Retrieves BRC-20 token balances for a Bitcoin address.

### Query parameters

<Property required={false} deprecated={false} name={"ticker"} type={"array of string"}>

</Property>

<Property required={false} deprecated={false} name={"block_height"} type={"string"}>

Bitcoin block height

<span>Pattern: `"^[0-9]+$"`</span>

</Property>

<Property required={false} deprecated={false} name={"offset"} type={"integer"}>

Result offset

<span>Minimum: `0`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>

### Path parameters

<Property required={true} deprecated={false} name={"address"} type={"string"}>

Bitcoin address

</Property>

| Status code | Description      |
| :---------- | :--------------- |
| `200`       | Default response |

</APIInfo>

