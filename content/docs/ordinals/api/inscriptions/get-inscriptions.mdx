Retrieves a list of inscriptions with options to filter and sort results.

### Query Parameters

<Property required={false} deprecated={false} name={"genesis_block"} type={"Any properties in string, string"}>

Bitcoin block identifier (height or hash)

</Property>

<Property required={false} deprecated={false} name={"from_genesis_block_height"} type={"string"}>

Bitcoin block height

<span>Pattern: `"^[0-9]+$"`</span>

</Property>

<Property required={false} deprecated={false} name={"to_genesis_block_height"} type={"string"}>

Bitcoin block height

<span>Pattern: `"^[0-9]+$"`</span>

</Property>

<Property required={false} deprecated={false} name={"from_genesis_timestamp"} type={"integer"}>

Block UNIX epoch timestamp (milliseconds)

</Property>

<Property required={false} deprecated={false} name={"to_genesis_timestamp"} type={"integer"}>

Block UNIX epoch timestamp (milliseconds)

</Property>

<Property required={false} deprecated={false} name={"from_sat_ordinal"} type={"integer"}>

Ordinal number that uniquely identifies a satoshi

<span>Minimum: `0`</span>

</Property>

<Property required={false} deprecated={false} name={"to_sat_ordinal"} type={"integer"}>

Ordinal number that uniquely identifies a satoshi

<span>Minimum: `0`</span>

</Property>

<Property required={false} deprecated={false} name={"from_sat_coinbase_height"} type={"string"}>

Bitcoin block height

<span>Pattern: `"^[0-9]+$"`</span>

</Property>

<Property required={false} deprecated={false} name={"to_sat_coinbase_height"} type={"string"}>

Bitcoin block height

<span>Pattern: `"^[0-9]+$"`</span>

</Property>

<Property required={false} deprecated={false} name={"from_number"} type={"integer"}>

Inscription number

</Property>

<Property required={false} deprecated={false} name={"to_number"} type={"integer"}>

Inscription number

</Property>

<Property required={false} deprecated={false} name={"id"} type={"array of string"}>

Array of inscription IDs

</Property>

<Property required={false} deprecated={false} name={"number"} type={"array of integer"}>

Array of inscription numbers

</Property>

<Property required={false} deprecated={false} name={"output"} type={"string"}>

An UTXO for a Bitcoin transaction

<span>Pattern: `"^[a-fA-F0-9]{64}:[0-9]+$"`</span>

</Property>

<Property required={false} deprecated={false} name={"address"} type={"array of string"}>

Array of Bitcoin addresses

</Property>

<Property required={false} deprecated={false} name={"genesis_address"} type={"array of string"}>

Array of Bitcoin addresses

</Property>

<Property required={false} deprecated={false} name={"mime_type"} type={"array of string"}>

Array of inscription MIME types

</Property>

<Property required={false} deprecated={false} name={"rarity"} type={"array of Any properties in string, string, string, string, string, string"}>

Array of satoshi rarity values

</Property>

<Property required={false} deprecated={false} name={"recursive"} type={"boolean"}>

Whether or not the inscription is recursive

</Property>

<Property required={false} deprecated={false} name={"cursed"} type={"boolean"}>

Whether or not the inscription is cursed

</Property>

<Property required={false} deprecated={false} name={"offset"} type={"integer"}>

Result offset

<span>Minimum: `0`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

Results per page

<span>Minimum: `1`</span>

<span>Maximum: `60`</span>

</Property>

<Property required={false} deprecated={false} name={"order_by"} type={"Any properties in string, string, string, string"}>

Parameter to order results by

</Property>

<Property required={false} deprecated={false} name={"order"} type={"Any properties in string, string"}>

Results order

</Property>

| Status code | Description      |
| :---------- | :--------------- |
| `200`       | Default Response |
| `404`       | Default Response |

</div>
