## Resumen

El Archivo Hiro te permite iniciar rápidamente servicios compatibles con datos precargados, lo que de otro modo podría llevarte días o semanas adquirir al sincronizar desde el génesis. Este es un servicio público que Hiro ofrece como un beneficio gratuito para la comunidad de Stacks.

[Ver el Archivo Hiro](https://archive.hiro.so/)
