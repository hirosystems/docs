## Resumen

The Chainhooks API is a REST API that allows you to programmatically manage chainhooks, configure event filters, and control webhook delivery. It provides the same functionality covered in the [Introducción al SDK de Chainhooks](/tools/chainhooks/introduction) through direct HTTP requests.

## Características Principales

* **Gestión completa de chainhook** - Crear, leer, actualizar y eliminar chainhooks
* **Configuración de webhook** - Set up HTTP POST endpoints for event delivery
* **Endpoints de evaluación** - Probar chainhooks contra bloques históricos

## URLs Base

La API de Chainhooks está disponible tanto en mainnet como en testnet:

```
Testnet: https://api.testnet.hiro.so/chainhooks/v1
Mainnet: https://api.mainnet.hiro.so/chainhooks/v1
```
