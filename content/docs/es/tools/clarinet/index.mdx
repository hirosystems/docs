## Visión general

Clarinet es la forma más rápida de construir, probar y desplegar contratos inteligentes en la blockchain de Stacks. Te proporciona un devnet local, REPL, marco de pruebas y herramientas de depuración para enviar código Clarity de alta calidad con confianza.

![Descripción general del clarinete](/images/tools/clarinet/overview.svg)

Para explorar las características de Clarinet con IA, copie y pegue [llms.txt](/tools/clarinet/llms.txt) en tu LLM de preferencia.

## Características principales

* [**Aprovecha una potente interfaz de línea de comandos**](/tools/clarinet/cli-reference) - Crea nuevos proyectos, gestiona tus contratos inteligentes y sus dependencias utilizando los requisitos de clarinet, e interactúa con tu código a través del REPL integrado.
* [**Escribir pruebas unitarias con el SDK**](/tools/clarinet/sdk-introduction) - Utiliza el SDK de Clarinet para escribir pruebas unitarias en un entorno JS familiar y validar el comportamiento del contrato.
* [**Ejecutar un entorno de blockchain privado**](/tools/clarinet/local-blockchain-development) - Inicia una red de desarrollo local con nodos, mineros y APIs para que puedas probar e integrar tu código.
* [**Extensión de VSCode**](/tools/clarinet/vscode-extension) - Linter, depurador paso a paso, ayuda a escribir contratos inteligentes (autocompletado, documentación, etc.).

## Instalación

<TerminalPicker storage="macOs">
  ```terminal !! macOS
  $ brew install clarinet
  ```

  ```terminal !! Windows
  $ winget install clarinet
  ```

  ```terminal !! Cargo
  $ sudo apt install build-essential pkg-config libssl-dev
  ```

  ```terminal !! Binary
  $ wget -nv https://github.com/hirosystems/clarinet/releases/latest/download/clarinet-linux-x64-glibc.tar.gz -O clarinet-linux-x64.tar.gz
  $ tar -xf clarinet-linux-x64.tar.gz
  $ chmod +x ./clarinet
  $ mv ./clarinet /usr/local/bin
  ```
</TerminalPicker>
