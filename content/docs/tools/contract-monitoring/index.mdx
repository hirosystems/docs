## Overview

Contract monitoring in the Hiro Platform lets you track smart contract events in real-time and receive webhook notifications when specific activities occur.

To explore Contract monitoring features with AI, copy and paste [llms.txt](/tools/contract-monitoring/llms.txt) into your LLM of choice.

## Key features

- ✓ **Real-time notifications** - Target specific events and receive webhooks within seconds
- ✓ **No-code setup** - Configure monitors through the Platform UI without writing code
- ✓ **Managed infrastructure** - No nodes, indexers, or databases to maintain
- ✓ **Analytics dashboard** - Visual insights into contract activity and trends
