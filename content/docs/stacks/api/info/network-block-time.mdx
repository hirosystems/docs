<API>

<APIInfo method={"GET"} route={"/extended/v1/info/network_block_times"}>

Retrieves the target block times for mainnet and testnet. The block time is hardcoded and will change throughout the implementation phases of testnet.

### Parameters

No parameters.

| Status code | Description |
|:------------|:------------|
| `200`       | Success     |

</APIInfo>

<APIExample>

```bash title="Terminal"
curl -L \
  "https://api.mainnet.hiro.so/extended/v1/info/network_block_times" \
  -H 'Accept: application/json'
```

<Tabs items={["200"]}>

<Tab value={"200"}>

GET request that returns network target block times

```json title="Example response"
{
  "mainnet": {
    "target_block_time": 0
  },
  "testnet": {
    "target_block_time": 0
  }
}
```
