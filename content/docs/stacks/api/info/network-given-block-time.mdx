<API>

<APIInfo method={"GET"} route={"/extended/v1/info/network_block_time/{network}"}>

Retrieves the target block time for a given network. The network can be mainnet or testnet. The block time is hardcoded and will change throughout the implementation phases of testnet.

### Path parameters

<Property required={true} deprecated={false} name={"network"} type={"string"}>

the target block time for a given network (testnet, mainnet).

<span>Example:  `"mainnet"`</span>

<span>Value in:  `"testnet" | "mainnet"`</span>

</Property>

| Status code | Description |
| ----------- | ----------- |
| `200` | Success |

</APIInfo>

<APIExample>

```bash title="Terminal"
curl -L \
  "https://api.mainnet.hiro.so/extended/v1/info/network_block_time/mainnet" \
  -H 'Accept: application/json'
```
