<API>

<APIInfo method={"GET"} route={"/v1/names/{name}/zonefile/{zoneFileHash}"}>

Retrieves the historical zonefile specified by the username and zone hash.

### Path parameters

<Property required={true} deprecated={false} name={"name"} type={"string"}>

Fully-qualified name

<span>Example:  `"muneeb.id"`</span>

</Property>

<Property required={true} deprecated={false} name={"zoneFileHash"} type={"string"}>

Zone file hash

<span>Example:  `"b100a68235244b012854a95f9114695679002af9"`</span>

</Property>

| Status code | Description |
| :----------- | :----------- |
| `200` | Success |
| `400` | Error |
| `404` | Error |

</APIInfo>

<APIExample>

```bash title="Terminal"
curl -X GET "https://api.mainnet.hiro.so/v1/names/muneeb.id/zonefile/b100a68235244b012854a95f9114695679002af9"
```
