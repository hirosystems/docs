To run a local devnet, run the `clarinet devnet start` command.

 ```console title="Terminal"
clarinet devnet start
```

<Callout title="Common errors" type="warn">
  <p>If you are getting `error: clarinet was unable to create network`, make sure you have Docker (or a similar service) installed and running locally. Check out the [Docker installation guide](/guides/installing-docker) for more details.</p>
</Callout>
