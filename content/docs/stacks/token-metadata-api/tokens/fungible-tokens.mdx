Retrieves a list of fungible tokens.

### Query Parameters

<Property required={false} deprecated={false} name={"name"} type={"string"}>

The name of the token

</Property>

<Property required={false} deprecated={false} name={"symbol"} type={"string"}>

The symbol of the token

</Property>

<Property required={false} deprecated={false} name={"address"} type={"string"}>

Stacks Address

<span>Example:  `"SP3K8BC0PPEVCV7NZ6QSRWPQ2JE9E5B6N3PA0KBR9"`</span>

<span>Pattern:  `"^[0123456789ABCDEFGHJKMNPQRSTVWXYZ]{28,41}"`</span>

</Property>

<Property required={false} deprecated={false} name={"offset"} type={"integer"}>

Result offset

<span>Minimum:  `0`</span>

</Property>

<Property required={false} deprecated={false} name={"limit"} type={"integer"}>

Results per page

<span>Minimum:  `1`</span>

<span>Maximum:  `60`</span>

</Property>

<Property required={false} deprecated={false} name={"order_by"} type={"Any properties in string, string"}>

Parameter to order results by

</Property>

<Property required={false} deprecated={false} name={"order"} type={"Any properties in string, string"}>

Results order

</Property>

| Status code | Description     |
| :---------- | :-------------- |
| `200`       | Default response|

</div>

```bash title="Terminal"
curl -X GET "https://api.hiro.so/metadata/v1/ft"
```
