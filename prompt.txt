## Authentication & Rate Limits

1. API Keys
   - Optional but recommended for higher rate limits
   - Generate free API keys at https://platform.hiro.so
   - Must be passed in header as 'x-api-key'
   - Should only be used server-side, never client-side

2. Rate Limits
   - Without API key: 50 RPM per client IP
   - With API key: 500 RPM regardless of IP
   - Limits apply across all APIs combined
   - Implement retry logic and rate limit handling

   - Best for: Account info, transactions, smart contracts

   - Authentication: Optional API key
   - Rate Limits: Shared across all APIs

## Standard Implementation Patterns

1. Using Fetch (Preferred for most APIs)
```typescript
async function makeRequest(url: string, options: RequestInit = {}) {
  const headers = {
    'Content-Type': 'application/json',
    'Accept': 'application/json',
    ...options.headers,
  };

  try {
    const response = await fetch(url, {
      ...options,
      headers,
    });

    if (response.status === 429) {
      const retryAfter = response.headers.get('Retry-After');
      await new Promise(resolve => 
        setTimeout(resolve, retryAfter ? parseInt(retryAfter) * 1000 : 60000)
      );
      return makeRequest(url, options);
    }

    return response.json();
  } catch (error) {
    console.error('API request failed:', error);
    throw error;
  }
}
```

2. Using Stacks API Client
```typescript
import { createClient, OperationResponse } from '@stacks/blockchain-api-client';

const stacksClient = createClient({
  baseUrl: 'https://api.mainnet.hiro.so'
});

// Example: Get mempool stats with typing
async function getMempoolStats(): Promise<OperationResponse['/extended/v1/tx/mempool/stats']> {
  const { data } = await stacksClient.GET('/extended/v1/tx/mempool/stats');
  return data;
}

// Example: Get block transactions
async function getBlockTransactions(heightOrHash: string | number, limit = 20, offset = 0) {
  const { data } = await stacksClient.GET('/extended/v2/blocks/{height_or_hash}/transactions', {
    params: {
      path: { height_or_hash: heightOrHash },
      query: { limit, offset }
    }
  });
  return data;
}
```

## Common Integration Patterns

1. Cross-API Integration Example
```typescript
import { createClient } from '@stacks/blockchain-api-client';

async function getAddressDetails(btcAddress: string, stxAddress: string) {
  // Stacks API using official client
  const stacksClient = createClient({
    baseUrl: 'https://api.mainnet.hiro.so'
  });
  
  // Other APIs using fetch
  const [stacksData, ordinalsData] = await Promise.all([
    stacksClient.GET('/extended/v2/address/{address}', {
      params: { path: { address: stxAddress } }
    }),
    fetch(`https://api.hiro.so/ordinals/v1/inscriptions?address=${btcAddress}`, {
      headers: {
        'Accept': 'application/json'
      }
    }).then(res => res.json())
  ]);

  return {
    stacks: stacksData.data,
    ordinals: ordinalsData
  };
}
```
